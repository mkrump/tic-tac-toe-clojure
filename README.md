# TicTacToe
Clojure version of TicTacToe

## Prerequisites
- Install [Leiningen](https://leiningen.org/)

- Clone the repo

	```
	git clone https://github.com/mkrump/tic-tac-toe-clojure.git
	```

## Building the project
```
TBU
```

## Running the tests
```
lein test
```

## Play TicTacToe
```
lein run
```
